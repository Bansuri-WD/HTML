Definatios

1.WebSite

A collection of web pages which are grouped together and usually connected together in various ways. Often called a "web site" or a "site."

2. WWW

www stands for world wide web. It is a system of creating, organizing and linking of documents. The documents on web are created in hyper text formate. Hypertext facilitates linking of documents.

3. HTTP/ HTTPS

http stand for hyper text transfer protocol. https stands for hyper text transfer protocol secured. It is fullfils clients request.

4.Client Server

Client-server is a relationship in which one program (the client) requests a service or resource from another program (the server). A client usually does not share any of its resources, but it requests content or service from a server.Examples of computer applications that use the clientâ€“server model are email, network printing, and the World Wide Web.

5. Domain

A domain is an address which allows a visitor to easily find the websites online.

6. Host

a host is a hardware that has a capability of permitting access to a network. A host works as a server offering information resources, services, applications to users or other hosts on the network.

7. Hosting

A hosting is a place where files are stored, it's also known as web site hosting, web hosting and web hosting.

8. Path

A path is a string of characters used to uniquely identify a location in a directory structure

9. Extension

Extensions tell your computer which application created or can open the file and which icon to use for the file. For example, the docx extension tells your computer that Microsoft Word can open the file and to display a Word icon when you view it in File Explorer.

10. URL - URI - URN

URI stands for Uniform Resource Identifier. URL stands for Uniform Resource Location. URN stands for Uniform Resource Name. URI identifies the resource (text document, image file, etc).URL is the subset of URI that includes a network location. URN is a subset of URI that includes a name within a given space, but no location.

11. Address Bar

An address bar is also known as a uniform resource locator (URL) bar, location bar or address box.
